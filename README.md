### 第八章

连续子向量最大和问题的优化

普通方法: n^3  立方算法

平方算法: n^2:   x[0,i] = x[0,i-1]  + x[i]   减少部分计算

​				或者先用一个额外数组记录累加和,减少计算

分治算法: 左半边最大和右半边最大,同时要考虑跨两边的情况   nlogn

线性算法:  n时间复杂度,使用扫描,以及比较记录最大值
